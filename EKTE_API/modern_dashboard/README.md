# Modern Dashboard for EKTE_API

Dette er en ny, ren mappe for all videre frontend- og dashboardutvikling til EKTE_API.

## Struktur
- Alle nye HTML, JS, CSS og eventuelle bilder eller komponenter skal ligge her.
- Ingen gamle eller eksperimentelle filer blandes inn.

## Første steg
1. Bygg en enkel index.html som kobler seg til API-et og viser systemstatus.
2. Utvid med visning av anlegg, fartøy, helsedata og søk.
3. Legg til moderne design og eventuelt kart/visualisering etter behov.

## Kjøring
- Åpne index.html direkte i nettleser (API må kjøre på http://127.0.0.1:8000)

---

**Alt nytt frontend-arbeid samles her for å unngå rot og gamle feil.**
