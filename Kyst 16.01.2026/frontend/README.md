# Kyst Monitor Phase 5 UI Agent

Moderne React + TypeScript dashboard for Kyst Monitor.

## Kom i gang

1. Gå til `frontend/`-mappen:
   ```bash
   cd frontend
   ```
2. Installer avhengigheter:
   ```bash
   npm install
   ```
3. Start utviklingsserver:
   ```bash
   npm run dev
   ```

- Åpne [http://localhost:5173](http://localhost:5173) i nettleseren
- API-kall mot `/api` og `/static` proxes automatisk til FastAPI-backend

## Videre arbeid
- Legg til Material UI eller Tailwind CSS for profesjonell styling
- Implementer dashboard, filtrering, kart, tabs og API-integrasjon
- Se `src/App.tsx` for startpunkt
