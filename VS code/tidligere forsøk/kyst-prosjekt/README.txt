Dette er den nye prosjektmappen for Kyst-prosjektet. Her finner du alle viktige filer, og du kan enkelt starte serveren herfra.

For å starte serveren:
1. Åpne terminalen.
2. Skriv: cd "C:\Users\janin\OneDrive\Skrivebord\kyst-prosjekt"
3. Skriv: python -m http.server 8080

Åpne så http://localhost:8080/ i nettleseren.

Alle datafiler legges i mappen data/.
Frontend-filer (admin.html, frontend.html, vessels.html) ligger direkte i denne mappen.
