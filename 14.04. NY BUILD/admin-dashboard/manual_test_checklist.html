📝 Notater og Funn

Bruk browser console (F12) for å se detaljerte feilmeldinger. Vanlige problemer:

  • Element ikke funnet: ID mismatch mellom HTML og JS
  • API 404: Feil endpoint-sti
  • API 500: Backend-feil
  • Tom tabell: Rendering-logikk feiler eller ingen data i respons
  • Knapp virker ikke: Event listener ikke koblet til
